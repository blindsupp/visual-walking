Attention Map Vizualizations

We train a ViT backbone and vizualize the attention maps on stepping stones. Observe that the network focuses attention on edges of the stones where the robot is about to step. This is evidence for the claim that the network has learnt some version of foothold planning.

The top image is the input depth image and the second image contains the attention mask from the last ViT layer overlayed on the depth image.


Videos with Same Mount

We design and 3d print a new mount for the camera that protects it in the case of a fall. We show that a single policy is able to do all terrain.
